Guide to LSA

This website provides a comprehensive guide on setting up and running trialwise analyses with least squares all (LSA) GLM. This guide assumes that you have a basic understanding of what LSA is. If this is a completely new concept to you, you should begin your journey by doing some reading (I link some papers and pages and papers).
